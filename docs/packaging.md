# Packaging VeriFrame

VeriFrame packages three pieces:

1. Angular UI
2. Tauri desktop shell
3. Python sidecar engine

## Build order

```powershell
pnpm --filter @veriframe/ui-angular build
pnpm --filter @veriframe/desktop-tauri build
```

## Sidecar policy

The Python engine must bind only to `127.0.0.1` and must require the session token generated by Tauri.

Do not expose the engine over LAN. That would be less “desktop app” and more “local evidence buffet.”
